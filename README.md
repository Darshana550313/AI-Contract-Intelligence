# AI-Powered Contract Intelligence & Risk

## 📖 Project Overview
AI-Powered Contract Intelligence & Risk is an NLP-based system that analyzes legal contracts, extracts important clauses, identifies potential risks, and provides intelligent insights using OCR, Machine Learning, and Vector Databases.

## 🎯 Objectives
- Extract text from contract PDFs
- Identify important legal clauses
- Detect potential risks in contracts
- Store contract embeddings in a vector database
- Provide API-based contract analysis

## 🛠️ Technologies Used
- Python
- FastAPI
- OCR (Tesseract, PyMuPDF)
- NLP
- Vector Database (Pinecone/Milvus)
- Git & GitHub
